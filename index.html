Шпаргалка Flexbox

Даний документ створений для швидкого візуального та частково теоретичного ознайомлення з властивостями Flexbox (CSS Flexible Box Layout Module).

Базові значення та термінологія

Flexbox - це CSS модуль, що включає в себе елементи з певним набором властивостей. Деякі з них використовуються для контейнера (батьківський елемент, що прийнято називати "flex-контейнер"), в той час, як інші використовуються для дочірніх елементів "flex-елементи".

Елементи в контейнері розташовуються або в напрямку головної осі (main axis від main-start до main-end) або в напрямку поперечної осі (cross axis від cross-start до cross-end). Напрямки цих осей можуть змінюватись в залежностві від значення flex-direction

main axis - головна вісь flex-контейнера уздовж якої розташовуються flex-елементи. Ця вісь може бути горизонтальною або вертикальною, в залежності від значення властивості flex-direction.

main start | main end - flex-елементи розміщаються в контейнері починаючи з main-start та закінчуючи main-end.

cross axis - вісь перпендикулярна до головної осі (main axis). Її напрямок залежить від напрямку головної осі.

cross start | cross end - flex-ряди заповнюються елементами та поміщаються в контейнер, починаючи від cross start flex-контейнера у напрямку до cross end.

Властивості flexbox можна поділити на дві групи, одна для flex-контейнера, а інша для flex-елементів. Також властивості можна поділити на: напрямок - вирівнювання - розмір. В даному документі розглянемо перший варіант.


Властивості flexbox для контейнера

display

Для створення flexbox-контейнера необхідно задати правило display: flex; або display: inline-flex; для батьківського HTML-елемента

Детальніше

Після створення flex-контейнера за допомогою властивості display: flex; або display: inline-flex; усі безпосередні дочірні елементи автоматично стають flex-елементами. Значення за замовчуванням для деяких початкових CSS властивостей вже визначені, тому після створення flex-контейнера усі flex-елементи, що містяться в ньому будуть поводитись певним чином:

  • Елементи відображаються в ряд (властивість flex-direction за замовчуванням має значення row)
  • Позиціонування елементів починається від початку Головної осі (main axis)
  • Елементи не розтягуються по основній осі, але можуть стискатися
  • Елементи розтягуються, щоб заповнити простір поперечної осі (cross axis)
  • Властивості flex-basis задано значення auto
  • Властивості flex-wrap задано значення nowrap

display: flex; - контейнер стане блочного рівня та займатиме весь доступний горизонтальний простір


display: inline-flex; - контейнер стане інлайнового рівня та буде займати простір залежний від розміру елементів.

Flex-контейнер не є звичним блочним чи інлайновим контейнером, тому деякі CSS властивості не застосовуються в контексті flex розміщення. Зокрема:

  • float і clear не створюють/забороняють "плавання" елемента та не виносять елементи з потоку;
  • css властивість columns не впливає на flex-контейнер;
  • псевдоелементи ::first-line та ::first-letter не застосовуються до flex-контейнера;
  • vertical-align не впливає на flex-елементи.

flex-direction

Ця властивість вказує, як flex-елементи розміщаються у flex-контейнері шляхом задання напрямку головної осі (main axis) flex-контейнера. Вони можуть розміщатись в двох напрямках, горизонтально, як рядки та вертикально, як колонки.

Детальніше

Для потрібного розміщення flex-елементів потрібно мислити з точки зору двох осей - головної осі (main axis) та побічної осі (cross axis). Головна вісь визначається властивістю flex-direction, а побічна вісь проходить перпендикулярно.

  • row (значення за замовчуванням): елементи розташовуються зліва направо (в ltr контексті та зправа наліво в rtl)
  • row-reverse - елементи розташовуються справа наліво ltr, зліва направо в rtl
  • column - елементи розташовуються зверху вниз
  • column-reverse - елементи розташовуються знизу вгору









1
2
3
.flex-container {
display: flex;
flex-direction: row;
}

flex-wrap

Властивість flex-wrap керує тим, як flex-контейнер буде розміщати свої елементи - в один або декілька рядків, а також напрямок розміщення рядків.

Детальніше

За замовчуванням flex-елементи будуть намагатися вміститися в один рядок. При необхідності їм можна дозволити переноситись на новий рядок

  • nowrap - значення замовчуванням, при якому всі flex-елементи будуть вибудовуватися в одну лінію.
  • wrap - flex-елементи будуть переноситись на кілька рядків, від верху до низу.
  • wrap-reverse - flex-елементи будуть переноситись на кілька рядків від низу до верху.



Flex-елементи відображаються в один ряд та звужуються, щоб уміститися в ширину flex-контейнера.

Flex-елементи відображаються в кілька рядів, зліва направо і зверху вниз

Flex-елементи відображаються в кілька рядів, зліва направо і знизу вгору.

Ці властивості залежать від режиму напрямку письма, тому в rtl-контексті вони відповідно будуть перевернуті.

justify-content

Властивість justify-content вирівнює flex-елементи відносно основної осі (main axis).

Детальніше
  • flex-start (за замовчуванням) - елементи розміщені на початку основної осі
  • flex-end - елементи розміщені в кінці основної осі
  • center - елементи центровані по основній осі
  • space-between - елементи рівномірно розподілені по лінії, перший елемент "прибитий" до початку рядка, останній елемент в кінці
  • space-around - елементи рівномірно розподілені по лінії з однаковим простором навколо них.
  • space-evenly - елементи розподіляються таким чином, щоб відстань між будь-якими двома елементами та відстань до країв були однаковими.

.flex-container {
display: flex;

align-items

Властивість align-items вирівнює flex-елементи вздовж поперечної осі (cross axis), перпендикулярно до основної осі (main axis).

Детальніше
  • stretch (за замовчуванням) - елементи розтягуються та заповнюють всю висоту поперечної осі (cross axis) flex-контейнера
  • flex-start - елементи розміщуються на початку поперечної осі
  • flex-end - елементи розташовуються в кінці поперечної осі
  • center - елементи розташовуються по центру по поперечної осі
  • baseline - елементи вирівнюються по базовій лінії

display: flex;
align-items: stretch;

align-content

Властивість align-content вирівнює рядки flex-контейнера по поперечній осі (cross axis).

Детальніше
  • stretch (за замовчуванням) - ряди розтягуються та заповнюють всю висоту поперечної осі
  • flex-start - ряди flex-елементів розміщуються на початку поперечної осі flex-контейнера
  • flex-end - ряди flex-елементів розміщуються в кінці поперечної осі flex-контейнера
  • center - ряди flex-елементів розміщуються по центру поперечної осі flex-контейнера
  • space-between - ряди flex-елементів відображаються з рівним інтервалом між ними, перший та останній ряд вирівнюється по краях flex-контейнера.
  • space-around - flex-елементи відображаються з рівним простором навколо кожного ряду flex-елементів.
  • space-evenly - елементи розподіляються таким чином, щоб відстань між рядами та відстань до країв були однаковими

Властивості для flexbox елементів

align-self

Властивість align-self перевизначає вирівнюванння за замовчуванням чи те значення, що вказане в align-items для flex-елементів.

Детальніше
  • auto (за замовчуванням) - приймає значення align-items flex-контейнера
  • flex-start - елементи розміщуються на початку поперечної осі
  • flex-end - елементи розташовуються в кінці поперечної осі
  • center - елементи розташовуються по центру по поперечної осі
  • baseline - елементи вирівнюються по базовій лінії контейнера
  • stretch - елементи розтягуються та заповнюють всю висоту поперечної осі (cross axis) flex-контейнера

Flex grow

Властивість flex-grow вказує на скільки буде ростягнутим flex-елемент відносно інших flex-елементів в контейнері при розподілі вільного простору. Значення за замовчуванням: 0. Від'ємні значення ігноруються.

Детальніше

Формула за якою відбувається розрахунок розтягування елементів виглядає наступним чином:





Нова ширина елемента = ((flex-grow елемента / сума значень flex-grow) * доступний вільний простір) + початкова ширина елемента


Якщо 3 елементу на зображенні задати значення flex-grow: 1;, то його ширина буде розраховуватись наступним чином:




Якщо 1 елементу задати значення flex-grow: 1;, a 3 елементуflex-grow: 3;то його ширина буде розраховуватись наступним чином:



Оскільки властивість flex-grow приймає цілочисельні значення, в нього можна записати і 0. Це означатиме, що flex-елемент не буде змінювати ширину за умови, що загальна ширина контейнера та ширина області перегляду сторінки буде рівна/більша за ширину елемента.

1
2
3
.flex-container {
display: flex;
}
.item-1 {
flex-grow: 0;
}
.item-2 {
flex-grow: 0;
}
.item-3 {
flex-grow: 0;
}

flex-shrink

Властивість flex-shrink визначає наскільки буде звужений flex-eлемент відносно інших flex-елементів в контейнері при розподілі негативного вільного простору.

Детальніше

Якщо змінити значення flex-shrink на 0 (не звужуватись), то він збереже свій відносний розмір. Значення за замовчуванням: 1. Від'ємні значення ігноруються.

Елемент зберігатиме свою ширину за умови, що:

  • загальна ширина всіх flex-елементів буде меншою за ширину контейнера;
  • ширина області перегляду сторінки буде рівна/більша за ширину елемента.
1
2
3
.flex-container {
display: flex;
}
.item-1 {
width: 100px;
} .item-1 {
flex-shrink: 1;
}
.item-2 {
flex-shrink: 1;
}
.item-3 {
flex-shrink: 1;
}

flex-basis

Властивість flex-basis вказує початковий розмір flex-елемента, перш ніж вільний простір розподілиться з іншими факторами контейнера чи елементів.

Детальніше

Властивість flex-basis приймає ті ж значення, які можуть приймати властивості width і height. Значення за замовчуванням: auto, яке встановлюється на основі значення content. Значення content задається автоматично, на основі розміру вмісту flex-елемента.

1
2
3
.flex-container {
display: flex;
}
.item {
width: 100px;
}
.item-3 {
flex-basis: auto;
}

flex

Властивість flex - це скорочений запис для властивостей flex-grow, flex-shrink та flex-basis. Значення за замовчуванням: initial, що відповідає значенню 0 1 auto.

flex: <flex-grow> <flex-shrink> || <flex-basis>

1
2
3
.flex-container {
display: flex;
}
.item {
flex: 0 1 auto;
}
.item-3 {
flex: 1;
}

order

Властивість order визначає порядок в якому будуть розміщені flex-елементи в контейнері. Значення елементів за замовчуванням 0, розміщаються один за одним в порядку їх додавання

1
2
3
.flex-container {
display: flex;
}
.item-1 {
order: 0;
}
.item-2 {
order: 0;
}
.item-3 {
order: 0;
}
